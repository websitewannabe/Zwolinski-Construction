Rebuilt Zwolinski Construction website from React 18 + Vite + React Router to Next.js 15 App Router.

**Why:** Client wanted modern framework with better SEO (server components, auto sitemap/robots) and upgraded luxury design.

**How to apply:** The site uses Next.js App Router, Tailwind CSS with custom gold (#C9A84C) accent on black backgrounds, Playfair Display serif headings, Inter body font. Deployed on Netlify with @netlify/plugin-nextjs. Contact form uses Netlify Forms. GTM tracking (GTM-M5W2MT4T) is in the root layout.
